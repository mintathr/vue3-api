instalasi dependency/ configurasi vue js3

1. upgrade cli ke last version => npm install -g @vue/cli
2. initial vite, ke folder projectnya (htdocs)=> npm init @vitejs/app <project-name> --template vue
3. masuk ke directory folder cd <project-name>(akun-client)
4. npm install
5. npm run dev
6. instal bs-5 => npm install bootstrap
7. instal popper dari https://www.npmjs.com/package/@popperjs/core 
	=> npm i @popperjs/core
8. npm install vue-router@next axios
